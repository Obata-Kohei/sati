# sati: Sati is an Analysis Toolset for Impact

This project compiles functions to calculate satellite orbits and information about the geomagnetic field in orbit.

We use `uv` as a management tool for Python and packages.

After `git clone`, you can use `uv sync` to build Python environment such as package installation.

`uv sync` to setup python environment
